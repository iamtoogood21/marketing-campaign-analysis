
# 📊 Marketing Campaign Analysis

## 🚀 Project Overview
This project predicts whether a customer will respond to a marketing campaign using Machine Learning.

## 🧠 Model
- Logistic Regression

## 📊 Features
- Age
- Income
- Previous Purchases
- Website Visits
